# Skill Usage Policy

External AI skills may be used when they match the task and have been installed intentionally.

## Rules

- Install external skills manually.
- Review skill instructions before relying on them.
- Do not treat external skills as project policy unless documented.
- Record durable workflow changes in `docs/harness/` or `docs/knowledge/`.

## Exclusions

This common template does not bundle external everything-claude-code skills or configure automatic skill installation.
